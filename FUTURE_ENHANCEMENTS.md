# Future Enhancements - BlueWings Airlines

This document tracks potential features that are out of scope for the current MVP but would add business and user value in future phases.

- Seat selection (visual layout selection inside chat)
- Baggage allowance and paid add-ons
- Real payment gateway integration (Stripe, Razorpay, etc.)
- Multi-city flight booking & group reservation support
- OTP verification via real SMS or WhatsApp template messages
- Multi-lingual agent handoff routing
