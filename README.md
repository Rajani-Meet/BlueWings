# BlueWings Airlines - Conversational Booking & Servicing MVP

This is the MVP for the **BlueWings Airlines Conversational Booking & Servicing** service submitted for the **22North Product Engineering Challenge 2026**.

## Tech Stack
- **Frontend**: Next.js, TypeScript, PWA, Vanilla CSS
- **Backend**: Node.js, Express, TypeScript, Prisma, PostgreSQL
- **LLM**: Claude API (Haiku) with Keyword Fallback
- **Messaging Channel**: WhatsApp Cloud API + Next.js PWA Web Chat UI
